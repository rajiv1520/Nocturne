# Nocturne Landing Page

This repository contains the landing page website for **Nocturne**, a premium movie streaming application. The site serves as the official distribution point for the Nocturne Android APK.

## Features

- **Modern & Premium Design**: Built with HTML, CSS, and JavaScript.
- **Responsive Layout**: Designed to look great on desktop, tablet, and mobile devices.
- **Dynamic Animations**: Utilizes particles.js and custom CSS animations for a visually engaging experience.
- **Direct Download**: Provides a direct, secure download link for the Nocturne Android APK (`.apk` format).
- **Dark Theme Aesthetics**: Matches the premium, cinematic vibe of the Nocturne app.

## File Structure

- `index.html`: The main structural file containing the landing page content.
- `style.css`: Contains all styling, custom fonts (Outfit), and animations.
- `script.js`: Handles interactivity and the initialization of particles.js.
- `assets/`: Contains project images and the downloadable application file (`nocturne-v1.0.apk`).

## Setup and Usage

To view this landing page locally, you don't need any special build tools or servers. 

1. Clone or download this directory.
2. Open the `index.html` file directly in your preferred web browser.
3. For the best experience, you can also serve it via a local development server (e.g., using VS Code Live Server, or Python's `http.server`).

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- FontAwesome (Icons)
- Google Fonts (Outfit)
- particles.js (Background animation)
